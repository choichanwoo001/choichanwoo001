## **Development Philosophy**

<div align="center">

```mermaid
mindmap
  root((개발 철학))
    실용적 설계
      과도한 추상화 지양
      명확한 코드 구조
      유지보수성 우선
    데이터 중심
      AI/ML 배경 활용
      근거 기반 의사결정
      성능 지표 추적
    완성도 추구
      데모를 넘어선 프로덕트
      사용자 경험 고려
      운영 안정성 확보
```

</div>

**실용적 설계** — 과도한 추상화보다는 명확하고 유지보수 가능한 아키텍처  
**데이터 중심** — AI/ML 배경을 살린 데이터 기반 의사결정과 성능 최적화  
**완성도 추구** — 단순한 데모가 아닌 실제 서비스로 사용 가능한
